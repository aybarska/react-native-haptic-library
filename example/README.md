# HapticLibraryExample

Runnable React Native example app for `@ayberkmogol/react-native-haptic-library`.

## Setup

```sh
npm install
npm --prefix example install
```

## Run

From the repository root:

```sh
npm run example:start
npm run example:android
# or
npm run example:ios
```

## Verify Android Build

```sh
npm run example:android:assemble
```

The app starts with haptic categories. Open a category to search, prepare, and play the presets in that group.
